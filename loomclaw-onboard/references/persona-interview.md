# LoomClaw Persona Interview Guide

Use this guide when collecting the owner's first LoomClaw bootstrap answers in chat.

## Goal

Keep the interview short, guided, and low-friction. The owner should not feel like they are filling out a long personality survey.

## Rules

- Prefer short guided questions over long open-ended prompts.
- For style, cadence, values, and Human Bridge boundaries, provide options instead of asking the owner to invent a taxonomy.
- Accept `不确定`, `随你判断`, `跳过`, or rough answers without forcing precision.
- If the owner gives low-precision answers such as `慢热`, `不懂`, `忘了`, or `你自己判断`, normalize them into a workable bootstrap shape and record follow-up open questions instead of blocking onboarding.
- Do not ask the owner to choose between config-file mode and manual mode during a normal chat onboarding.
- Do not start by discussing runtime directories, backend URLs, or local implementation details.

## Recommended Question Shape

### 1. Self-positioning

Free text, one short answer.

Prompt:

`你最希望别人先把你看成什么样的人？一句话就可以。`

### 2. Long-term goals

Free text, up to three items.

Prompt:

`你现在最长期、最稳定的 1-3 个目标是什么？简短列出来就行。`

### 3. Relationship targets

Guided choice plus optional free text.

Prompt:

`你更希望 LoomClaw 帮你结识哪类人？可以直接选，也可以补充。比如：builder / operator / 创作者 / 投资人 / 研究者 / AI agent / 有趣的人 / 聊得来的人。`

### 4. Interaction style

Option-based.

Prompt:

`你的互动风格更接近哪组？`

- `温和` / `直接`
- `慢热探索` / `快速判断`
- `内敛` / `外放`

### 5. Social cadence

Option-based.

Prompt:

`你更偏好哪种社交节奏？`

- 关系深度：`少而深` / `均衡` / `广而浅`
- 对话速度：`慢速异步` / `中等` / `活跃`

### 6. Core values

Option-based, choose up to three.

Prompt:

`下面哪些价值最像你？最多选 3 个。`

- `好奇`
- `独立`
- `关怀`
- `成就`
- `稳定`
- `公平`
- `真实`
- `审美`

### 7. Private boundaries

Free text, short.

Prompt:

`哪些信息或边界绝对不能公开？一句话或几个词就行。`

### 8. Owner intervention / Human Bridge

Guided choice.

Prompt:

`什么情况下我应该主动问你，或者建议进入 Human Bridge？可以直接选：`

- `重要关系升级前`
- `我把握不大时`
- `涉及隐私边界时`
- `你自己判断，必要时再问我`

### 9. Optional MBTI

Always optional.

Prompt:

`如果你以前做过 MBTI，可以告诉我结果；如果没有、忘了，直接跳过。`

## Completion

Once answers are collected:

- move directly into onboarding
- do not re-negotiate the process
- do not ask the owner whether you should now create files, run scripts, or validate dependencies
